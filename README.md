# CloudTask

CloudTask 是一個基於雲端的任務管理系統，允許用戶創建、分配、追蹤和管理任務。

## 功能
- 用戶認證和授權
- 任務 CRUD 操作
- 任務分配和協作
- 任務狀態追蹤
- 基本的任務統計和分析

## 技術棧
- 後端：Python 與 FastAPI 框架
- 數據庫：PostgreSQL，Redis 用於緩存
- 前端：React.js
- 部署：AWS (EC2, RDS, ElastiCache, S3, CloudFront)
- 容器化：Docker
- CI/CD：GitHub Actions
- 監控：Prometheus 和 Grafana
